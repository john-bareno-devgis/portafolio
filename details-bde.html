Bases de Datos Espaciales (BDE)

La estructuración de una BDE garantiza la centralización y accesibilidad de su información geográfica, con tecnologias libres o comerciales se diseña y estructura la base de datos con un componente espacial que hace de sus datos información georreferenciada, accesible y estandarizada, ideal para poder usar sus datos en la creación de tableros de control, geovisores o formularios de recopilación de datos en campo.

  • Recopilación y alojamiento de la información geográfica
  • Diseño y estructuracion de la Base de datos relacional espacial
  • Diseño y creacion de formularios de captura de datos geograficos en campo
  • Integración con software SIG (ArcGIS, QGIS, etc)
  • Integración con software de tableros de control (Power BI, ArcGIS Insights, etc)
