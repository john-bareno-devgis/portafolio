Sobre mi

Soy ingeniero catastral y geodesta, apasionado por el desarrollo web y las tecnologías geoespaciales. Con más de 5 años de experiencia en el desarrollo e implementación de soluciones web geográficas.

Desarrollador Web Full Stack

A lo largo de mi experiencia profesional he tenido la oportunidad de trabajar como desarrollador en el nivel front-end y back-end, así como analista de datos geográficos, profesional experto en SIG, también he cubierto rol de líder de equipo coordinando proyectos de desarrollo SIG. Esta experiencia me ha permitido obtener capacidades para desarrollar aplicaciones web geográficas, implementar infraestructuras de datos espaciales, automatizar tareas de procesamiento de información geográfica, diseñar, implementar y administrar bases de datos geográficas.

Algunas de las tecnologías en las que me he capacitado y trabajado incluyen:

  • Angular
  • React
  • JavaScript
  • Python
  • HTML y CSS
  • ArcPy
  • Node.js
  • Express.js
  • PostgreSQL + PostGIS
  • GIT

Cuento tambien con conocimiento tecnico solido en el uso de software como ArcGIS Pro, QGIS, ArcGIS Desktop, Power BI, ArcGIS Insights, ArcGIS Dashboards, QField, GeoNode, GeoServer, QGIS Server, ArcGIS Enterprise.
